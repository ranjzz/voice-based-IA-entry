# Voice Recognition System - Quick Reference Guide

## Quick Start (3 Commands)

```bash
# 1. Navigate to project
cd "c:\Users\ranji\OneDrive\Desktop\voice based\voice-recognition-system"

# 2. Build and test
build.bat

# 3. Run application
mvn exec:java -Dexec.mainClass="com.voicerecognition.ui.VoiceRecognitionApp"
```

## Essential Commands

### Build & Compile
```bash
mvn clean compile      # Compile only
mvn clean package      # Build JAR
build.bat              # Automated build (Windows)
```

### Testing
```bash
mvn test                                          # Run all tests
mvn test -Dtest=ValidationManagerTest            # Run specific test
mvn test -Dtest=ValidationManagerTest#testValidateMarksValid  # Run specific method
```

### Running
```bash
mvn exec:java -Dexec.mainClass="com.voicerecognition.ui.VoiceRecognitionApp"
java -jar target/voice-recognition-system-1.0.0.jar
```

## Database Tables

### ia_marks
```
id (PRIMARY KEY)
roll_number (NOT NULL)
subject_name (NOT NULL)
marks (NOT NULL, 0-20)
faculty_name
timestamp (NOT NULL)
created_date
updated_date
```

## Troubleshooting Quick Links

| Issue | Solution |
|-------|----------|
| Maven not found | Install from maven.apache.org |
| Java not found | Install JDK 11+ from oracle.com |
| Microphone error | Check system audio settings |
| Database error | Delete .db file and restart |
| Build fails | Run `mvn clean install` |
